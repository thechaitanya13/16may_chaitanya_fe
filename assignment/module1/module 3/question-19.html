QUESTION-19 What is difference between HTML and XHTML?


XHTML

XHTML stands for Extensible Hypertext Markup Language.

XHtml is case sensitive language

Because of xhtml is a case sensitive, it will not run when it has an error.

HTML

HTML stands for Hypertext Markup Language

Html is not case sensitive.

The html is not case sensitive so it will run the program even if the program has error.
