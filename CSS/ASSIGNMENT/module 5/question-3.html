Q-3: Semantic element in HTML5?

Answer:There are some examples of semantic elements that are used in HTML5

  • Semantic HTML, also known as semantic markup

  • It refers to the use of HTML tags that convey the meaning—or semantics—of the content contained within them.

  • By adding semantic HTML tags to your pages, you provide additional information that helps define the roles and relative importance of the different parts of your page.
