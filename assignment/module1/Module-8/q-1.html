Q-1 What are the advantages of Bootstrap?

Ans.

  1. Easy to understand

    2. It is very easy to use. Anybody having basic knowledge of HTML ,CSS and Basic js can use Bootstrap.

  2. Responsive layout

    3. Bootstrap provides a responsive layout which for various screen (desktops, tablets, and mobile phones).

  3. Support all browser

    4. Bootstrap is compatible with the latest versions of all modern browsers and platforms.

  4. Pre-styled Components

    5. Bootstrap provides many Pre-styled Components ,developer need's just add bootstrap's classes.

  5. Reduce use of JavaScript plugins

    6. JavaScript plugins are bundled in the Bootstrap package.So adding these functionalities is easy as you just need to write a few lines of code and you are all set.
