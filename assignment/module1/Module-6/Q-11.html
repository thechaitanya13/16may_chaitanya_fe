Q-11 What are all the looping structures in JavaScript?

In javascript different loop like while loop, do while loop, for loop, for in loop, for of loop,

1.while loop:-

  • In JavaScript, the while loop is used to where require repeat code as long as a specified condition is true.
  • ex.

    let count = 0;

    while (count < 5) {

    console.log(count);

    count++;

    }

    • 2. do while loop:-

  • in javascript do while use check the condition after run one time code .
  • Ex.

    let count = 0;

    do {

    console.log("Count is: " ++count);

    count++;

    } while (count < 5);

    • 3.for loop

  • in for loop first check condition and after run code
  • Ex.

    for (let i = 0; i < 5; i++) {

    console.log(i);

    }

    • 4.for in loop

  • Ex.

    let person = {

    name: 'gaurav',

    age: 20,

    city: 'rajkot'

    };

    for (let i in person) {

    console.log(i + ': ' + person[i]);

    }

    • 4.for of loop

  • Ex.

    let a = ['apple', 'banana', 'orange'];

    for (let fruit of a) {

    console.log(fruit);

    }
