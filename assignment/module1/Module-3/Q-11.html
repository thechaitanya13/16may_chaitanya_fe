Que-11 What is the difference between a block-level element and an inline element?

Block-level element:-

  • In block-level element's width is set full screen's width.
  • it always start with new line in web-page.
  • Ex.by using this tag we can make block element, <div > ,<section > ,<li > , <p >.

    • Inline element:-

  • In inline element's width is set according to the content.
  • Ex.by using this tag we can make inline element , <span > ,< strong>,<b ><em >
