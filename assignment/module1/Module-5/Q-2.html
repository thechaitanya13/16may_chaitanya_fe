Q-2   How to embed audio and video in a webpage?

Ans.

  • In webpage audio is embed by <audio> tag and video is embed by <video> tag

    • Audio tag:-

  • In audio's start tag we write attribute for it and we put audio in <source> tag
  • in audio tag we use like this attribute, "Autoplay,control,loop,muted."

    • Video tag:-

  • In video's start tag we write attribute for it and we put audio in <source> tag
  • In video tag we use like this attribute,"Autoplay,control,loop,muted,height,width,Poster."
