Q-9 How can the style/class of an element be changed?

  • By javascript , 2 method we can change class of element ,hear example of both method.

    • 1. toggle method

  • Clicking the "toggle " button will toggle the red class on and off using classList.toggle('red').

    • 2. replace method

  • Clicking the "Replace " button will replace the green class with the coral class using classList.replace('green', 'coral').
