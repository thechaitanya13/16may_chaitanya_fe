Q-3.   What is the difference between CSS2 and CSS3?

Ans.

Css3 is latest version of css so it has more powerful feature than css-2

Hear difference between css-2 and css-3 on some bassis.

No. basis Css-2 Css-3
1 Selector type In Css2 we use selectors like element names,IDs, and class In Css3 we use Pseudo-classes, Pseudo-elements and descendant Selector.
2 Text For text we use in css2 font style,font weight,font size,text align etc. For text we use in css3 text shadow, text-overflow,font's color's gradient etc.
3 Background's Gradient in this we use solid color. in this we can use linear-gradient and radial-gradient
4 Element position In this use of margin and float for element position in web page In this use of Flex and Grid for layout creation .
5 Animation in css-2 there is not facility of Animation creation. in css-3 we can make different type beautiful Animation.
6 Media Queries By css2 we can not make responsive webpages By css3 we can make responsive webpages by use of media-queries
