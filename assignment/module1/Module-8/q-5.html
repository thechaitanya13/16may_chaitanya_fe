Q-5 What is the difference between Bootstrap 4 and Bootstrap 5 ?

Ans.

Bootstrap 4:-

  • Bootstrap 4 has 5 default responsive tiers namely xs, sm, md, lg, xl.
  • Responsive font size is not by default.
  • It has jquery and all related plugins.
  • It has limited color supports.

    • Bootstrap 5:-

  • It has 6 default responsive tiers namely xs, sm, md, lg, xl, xxl.
  • Responsive font size enabled by default
  • Jquery is removed and switched to vanilla JS with some working plugins.
  • It supports extra color with an expanded colors.
