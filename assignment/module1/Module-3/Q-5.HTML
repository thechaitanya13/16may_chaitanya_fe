Que-5 What are different types of lists in HTML? With Example.

List tag is use to show user's information inthe list format. There are main three types of lists:
1. Unordered List
2. Orderd List
3.Description List/data list

*Unordered List:-

Used when we don't need items in our ordered term.
In this we use ul and li. <ul>Unordered list</ul> <li>List Item</li>
Example:- classroom things

  • writing bord
  • Table
  • Marker
  • Benches

*Ordered List:-

Used when we need items in our ordered term. In this we use <ol >and &li;li >.
For example:- number

  1. One
  2. Two
  3. Three
